## What this is

Not a newsletter. Not financial advice. Just one person's notes from paying attention to markets for a long time — written in plain language, with full attribution to the sources that informed each piece.

Built with plain HTML, CSS, and minimal JavaScript. No frameworks, no build tools. Hosted on GitHub Pages.
